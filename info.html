What is Qmeet?

Qmeet is software for quickly setting up and conducting online meetings

setup meeting?

To setup meeting on Qmeet, goto the home page and click on the green icon with the video icon you will be redirected to the page with the meeting code copy the meeting code and head back to the home page to join the meeting

Join meeting?

to join a meeting on Qmeet, make sure you have the meeting code then head over to the home page and click on the orange button then enter the meeting code to join the meeting
